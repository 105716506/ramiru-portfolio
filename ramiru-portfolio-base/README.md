# Ramiru Portfolio — Base Version

Simple portfolio foundation built with HTML, CSS and JavaScript.

No npm, no frameworks, no package installation and no build process.

## Open the website

Simplest method: double-click `index.html`.

For development in VS Code, use the Live Server extension and open `index.html` with Live Server.

## Files

- `index.html` — page content and sections
- `styles.css` — neutral layout and responsive structure
- `script.js` — mobile navigation and automatic footer year
- `assets/images/` — profile/project images
- `assets/documents/` — CV PDF

## Update these placeholders

Search `index.html` for:

- `your-email@example.com`
- `href="#"`
- `Add project image`

Replace them with your real email, GitHub, LinkedIn, repository links and screenshots.

## Styling later

The CSS is intentionally neutral so you can later add:

- colour themes
- gradients
- glass effects
- custom typography
- hover states
- motion and scroll effects
